# Keyboard Shortcuts

##### i3wm htpc w/ Rii mini i8 keyboard

## General
* Win + V       = Clipboard History
* Win + Q       = Quit Application
* Win + T       = Open Terminal
* Win + Space   = Open App Launcher
* Search        = Open App Launcher

## Window management

### Focus
* Win + W       = Focus Up
* Win + A       = Focus Left
* Win + S       = Focus Down
* Win + D       = Focus Right
* Win + (1-0)   = Focus Desktop 1-10
* Win + Tab     = Focus Last

### Move
* AltGr + W     = Move Up
* AltGr + A     = Move Left
* AltGr + S     = Move Down
* AltGr + D     = Move Right
* AltGr + (1-0) = Move to Desktop 1-10
* AtlGr + Tab   = Move to Last

### Resize
* AltGr + Plus  = Increase
* AltGr + Minus = Decrease

### Scratchpad
* Win + Grave   = Show Scratchpad
* AltGr + Grave = Move to Scratchpad

### Other
* Win + F       = Float Window
* Win + D       = Dock Window
* AltGr + F     = Toggle Fullscreen
