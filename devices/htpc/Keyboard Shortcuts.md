# Keyboard Shortcuts

##### i3wm htpc w/ Rii mini i8 keyboard

## General
* Win + V       = Clipboard History
* Win + Q       = Quit Application
* Win + T       = Open Terminal
* Win + Space   = Open App Launcher
* Search        = Open App Launcher

## Window management

### Focus
* Win + W       = Focus Up
* Win + A       = Focus Left
* Win + S       = Focus Down
* Win + D       = Focus Right
* Win + (1-0)   = Focus Desktop 1-10
* Win + Tab     = Focus Last

### Move
* Alt + W       = Move Up
* Alt + A       = Move Left
* Alt + S       = Move Down
* Alt + D       = Move Right
* Alt + (1-0)   = Move to Desktop 1-10
* Alt + Tab     = Move to Last

### Resize
* Alt + Plus    = Increase
* Alt + Minus   = Decrease

### Scratchpad
* Win + Grave   = Show Scratchpad
* Alt + Grave   = Move to Scratchpad

### Other
* Win + F       = Float Window
* Win + D       = Dock Window
* Alt + F       = Toggle Fullscreen
